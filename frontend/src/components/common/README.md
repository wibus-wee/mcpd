# Components/Common

App-specific shared components used across multiple features.
These components contain app-specific logic but are reusable throughout the app.
Place universal UI primitives in `components/ui/` instead.

## Files

- **app-sidebar.tsx**: Main application sidebar with navigation menu
- **app-topbar.tsx**: Top bar with status indicator, theme toggle, and window controls
- **main-content.tsx**: Main content area wrapper with topbar integration
- **universal-empty-state.tsx**: Reusable empty/error state component with icon, title, description, and action
